========================================
服务器部署步骤:
========================================

1. 上传代码到服务器
   - 可以使用 scp, rsync 或 git

2. 配置环境
   cp config/config.example.yaml config/config.yaml
   vim config/config.yaml

3. 修复数据库（首次部署）
   cd scripts
   mysql -h <host> -P <port> -u <user> -p < fix_charset.sql
   cd ..

4. 编译程序
   go build -o admin-bot cmd/bot/main.go

5. 运行程序
   ./admin-bot
   
   或使用 systemd（推荐）
   详见 DEPLOY.md
